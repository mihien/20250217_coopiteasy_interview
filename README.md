## Installer les dépendances

J'ai utilisé poetry pour gérer l'environnement virtuel/les dépendances, mais j'ai généré un requirements.txt que ce soit installable facilement avec pip aussi.

## Comment lancer le projet

Depuis la racine du projet:
```
python 20250217_coopiteasy_interview/main.py
```
Un dossier output sera créé avec tout les fichiers html résultants, ainsi qu'un png avec le graphe du site
